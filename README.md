# 4K_60fps_ME

Motion Vector Estimation on 4K@60fps Video 运动向量估计硬件电路设计

## 1. 基本信息

### 1.1. 名词解释

- ME：Motion Estimation
- SAD：Sum of Absolute Difference
- SADPS：SAD per cycle
- X帧、X：当前帧
- Y帧、Y：先前帧
- 小块：8x8的X帧小块
- 大块：23x23的Y帧大块，即搜索区域
- 九宫格：大块的中心是小块，延长小块边得到大块的“井”字形分割，是大块的基本复用单位
- 小格：大块九宫格中的一格
- 大核：独立承担一个大块搜索任务的核心，一般几“核”指几个大核
- 小核：在大核内，负责进行256次SAD的运算单元

### 1.2. 常数速查

#### 1.2.1. 帧性质

- 4K：3840*2160@60fps
- 像素性质：8bit亮度
- 小块数量：480*270=129600
- 大块总信息量：23^2=529B

#### 1.2.2. 面积

- IO：35um X 110um

逻辑单元：高度均为1.8um，其中寄存器（时许逻辑单元）的宽度为5.6um，电路中使用频率最高的几种组合逻辑单元宽度如下（XN表示驱动强度）：

- INVX1：0.6um
- NANDX1：0.8um
- NORX1：0.8um

#### 1.2.3. 延时

55nm 工艺库常用cell延迟大约为：

- D flip-flop的Tclk-to-Q：~0.075ns
- 全加器 ADDF：~0.17ns
- NAND2: ~0.02ns
- IO：~2.5ns
